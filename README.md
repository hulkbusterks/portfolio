  # Portfolio (React + TypeScript + Vite)

  A small personal portfolio built with React, TypeScript and Vite. This repository contains the source, a Vite build, and a GitHub Actions workflow that builds and deploys the static `dist` output to GitHub Pages.

  **Quick Start**
  - **Prerequisites:** Node.js 18+, npm
  - **Install:**

    ```bash
    npm ci
    ```

  - **Dev:** Runs the dev server with HMR

    ```bash
    npm run dev
    ```

  - **Build:** Produces static files in `dist`

    ```bash
    npm run build
    ```

  - **Preview production build:**

    ```bash
    npm run preview
    ```

  **Project structure (high level)**
  - `src/` — app source code
  - `public/` — static public assets
  - `dist/` — generated by `npm run build` (not checked in)
  - `.github/workflows/deploy.yml` — GitHub Actions deploy workflow

  **Deploying manually**
  - Option A — Use the GitHub Actions workflow (recommended): push to `main` and the workflow will build and deploy.
  - Option B — Use `gh-pages` package to publish `dist`:

    ```bash
    npm run build
    npx gh-pages -d dist
    ```
